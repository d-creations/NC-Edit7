Favicon assets
================

This folder contains favicon assets for the NC-Edit7 application.

- `favicon.svg`: the primary SVG favicon used by the web app and FastAPI routes.
- `favicon-root.svg`: the previous root-level SVG kept as a local backup during the move.

To replace the favicon, update `favicon.svg` and optionally add PNG/ICO fallbacks in this folder. Modern browsers prefer `image/svg+xml` for crisp scaling.
